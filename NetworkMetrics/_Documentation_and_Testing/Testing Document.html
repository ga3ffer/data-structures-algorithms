Testing Document

 

This program takes a single Netdraw .VNA file as an command line argument.  It parses the information from the file line by line and creates a graph with the collected vertex and arc data specified by the .VNA file.  The program then collects and outputs basic statistics on the graph.

 

*This document does not include results for mean geodesic distance and graph diameter metrics 

 

Sample .vna files included

 

SCC-Test.vna

celegansneural.vna
